# Api-pattern-Standoff-2
Всем привет, в данном репозитории выложен исходный код api pattern или как называется в простонародий подсос патернов, я решил выложить две версии подсос с наклейкой и подсос без наклеек(то есть дефолтный),  вы можете пользоваться этим и радоваться написал именно для github не свою основную версию остальные боты в тгк: @Filyatrayder
Особенности данного pattern бота связаны с тем что:
1 - есть автопокупка нужного паттерна если его нету в инвентаре
2 - подставляет моментально
3 - вас не смогут задрейнить присувствует антиспам фильтр небольшой но рабочий
4 - можно лутать много голды
5 - возможен бан учитывая ныншнее время в Standoff 2
6 - удобное использование Telegram бота(можете кастомить под себя с помощью ИИ)
7 - больше о ботах и Api standoff находится в моем Telegram chanell @Filyatrayder
8 - для использования бота вам нужная библиотека Astandy(она есть в архиве, не уверен что положил точную версию Astandy но если что пишите поменяю)
9 - запуск python3 pattern.py
